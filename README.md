Aplicação para consulta de CEP por meio de uma API Web gratuita. Elaborada no intuito de praticar e aprimorar o desenvolvimento web.

Tecnologias utilizadas:
1 - HTML5;
2 - CSS3;
3 - Bootstrap 4;
4 - JavaScript (ES6);
5 - Ajax
